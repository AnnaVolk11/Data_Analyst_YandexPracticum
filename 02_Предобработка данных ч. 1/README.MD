Исследование надежности заемщиков

Анализ влияния семейного положения и количества детей на факт погашения кредита в срок.

Ход исследования:
1. Обзор данных.
2. Предобработка данных.
3. Проверка гипотез.

Полученные навыки:

2. Работа с дубликатами и пропусками.
